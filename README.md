# Docker команды

### Создание образа проекта
```bash
yarn create-image
```

### Запустить образ на сервере
```bash
yarn start-docker
```

# Scripts проекта

### Запуск дев разработки (локальный сервер) 
```bash
yarn dev
```

### Сборка проекта
```bash
yarn build
```
### Запуск сборки на локальном сервере
```bash
yarn start
```

### Проверка скриптов(JS) через ESLint
```bash
yarn lint
```

### Сборка шрифта иконок Icofont
```bash
yarn icofont
```
